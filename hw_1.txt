Первое ДЗ
Linux terminal (GitBash) commands

1) Посмотреть где я: pwd
2) Создать папку: mkdir group_25
3) Зайти в папку: cd group_25
4) Создать 3 папки: mkdir d_1 d_2 d_3
5) Зайти в любую папку: cd d_1
6) Создать 5 файлов (3 txt, 2 json): touch test_1.txt test_2.txt test_3.txt test_4.json test_5.json
7) Создать 3 папки: mkdir a_1 a_2 a_3
8) Вывести список содержимого папки:ls -la
9) + Открыть любой txt файл: vim test_1.txt
10) + написать туда что-нибудь, любой текст: наживаем i (INSERT) и пишем текст
11) + сохранить и выйти: esc :wq enter
12) Выйти из папки на уровень выше: cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку:cd group_25
                                                                        cd d_1
                                                                        mv test_1.txt test_4.json /c/Users/Pegova/group_25/d_2
14) скопировать любые 2 файла, которые вы создали, в любую другую папку: cp test_1.txt test_4.json /c/Users/Pegova/group_25/d_2
15) Найти файл по имени: find test_1.txt
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает: tail -f text_1.txt
17) вывести несколько первых строк из текстового файла: head -2 text_1.txt
18) вывести несколько последних строк из текстового файла: tail -2 text_1.txt 
19) просмотреть содержимое длинного файла (команда less) изучите как она работает: less text_1.txt
20) вывести дату и время: date
